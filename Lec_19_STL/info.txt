STL is broadly divided into four main components:

1. Containers
Used to store collections of data.

   Type	         Examples	                         Description
   ---	         -------	                         -----------
   Sequence	      vector, list, deque	             Linear, ordered collections
   Associative	   set, map, multiset, multimap	    Sorted key-based access
   Unordered	   unordered_set, unordered_map	    Hash tableâ€“based collections
   Adapters	      stack, queue, priority_queue	    Modify interface of other containers

2. Algorithms
Predefined functions for operations like:

   Searching (find)
   Sorting (sort)
   Merging (merge)
   Counting (count)
   Reversing (reverse)

3. Iterators
Act like pointers to traverse containers.

Types:

   InputIterator
   OutputIterator
   ForwardIterator
   BidirectionalIterator
   RandomAccessIterator

4. Function Objects (Functors)

Objects that can be called like functions.
Useful in custom comparisons during sorting, etc.
